# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] — 2026-07-03

### Added

- Project foundation and repository structure
- Next.js 16 with App Router, TypeScript strict mode, Tailwind CSS v4
- ESLint, Prettier, Husky, lint-staged, commitlint configuration
- Feature-based modular architecture
- Environment variable templates (local, preview, production)
- GitHub issue templates (bug report, feature request)
- GitHub pull request template
- Dependabot configuration for automated dependency updates
- CI/CD pipeline with lint, typecheck, build stages
- Security documentation and enterprise security standards
- Accessibility standards (WCAG AA)
- Coding standards and engineering guidelines
- Architecture Decision Records (ADR) tracking
- Documentation structure (architecture, security, API, database, etc.)
- Community files (README, LICENSE, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY)
