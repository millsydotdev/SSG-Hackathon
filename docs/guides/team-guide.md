# Team Guide

This guide covers team roles and responsibilities in SSG-Hackathon.

## Joining a Workspace

Team members join via invitation codes. Invitations are generated by the Owner or Team Lead.

## Team Roles

| Role | Permissions |
|------|------------|
| Owner | Full platform access, admin centre, all workspaces |
| Lead | Manage tasks, planning, team membership |
| Member | Standard participant — create and edit workspace content |
| Guest | Read-only access to workspace content |

## Member Capabilities

- Create and manage tasks, ideas, research, notes, files
- Participate in planning (objectives, milestones, deliverables)
- Submit deliverables
- View workspace analytics
- Set up GitHub integration (PAT or OAuth)
- Configure personal notifications

## Account Management

- Profile settings: name, username, timezone, social links
- Account settings: password change
- Security: active session monitoring
